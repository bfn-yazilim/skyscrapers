About Game

The Apartment Game is one of those easy-to-learn, addictive logic puzzles. This game doesn't require any math to solve, and instead requires only logic, and can be a source of entertainment for puzzle enthusiasts of all abilities and ages.

Rule 1: There cannot be a repeating number in each row and each column.

Rule 2: The numbers on the sides indicate the number of apartments visible when viewed from there. For example: If it says 4, it means you can see all the buildings. If the first building is 1-storey and the second building is 2-storey, you can see both the 1-storey and 2-storey buildings. Since it is in the order of 1,2,3,4, you can see all the buildings.

The goal is to fill each grid with the numbers 1 to 4. This stage of the game is similar to Sudoku. The basic rule is that each number must appear only once in each row and column.
